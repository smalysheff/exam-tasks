== Навигация проекту ==
Ctrl + E                Недавние файлы
Ctrl + Tab              Переключатель по файлам
Ctrl + Shift + E        Последние изменённые файлы
Alt + F7                Найти все места, где используется интерфейс / класс / метод (Ctrl + Click / Ctrl + B)
Alt + <-/->             Переход по вкладкам
F4                      Перейти к источнику
Alt + Home              Navigation bar

--Маркеры
F11                     Создать маркер
Shift + F11             Посмотреть маркеры
Ctrl + F11              Создать именованный маркер
Ctrl + markerName       Перейти к маркеру
Ctrl + Shift + F11      Посмотреть именованные маркер


== Навигация по классу ==
Ctrl + +/-              Свернуть и развернуть блоки кода
Ctrl + Shift + +/-      Свернуть и развернуть все блоки кода
Ctrl + Alt + +/-        Свернуть и развернуть все подблоки кода

Ctrl + F12              Структура класса (методы)
Ctrl + G                Перейти на строку №
Alt + ↓/↑               Следующий метод
F2                      Переход на место с ошибкой


== Работа с текстом / рефакторинг ==
Ctrl + W                Выделить слово
Ctrl + Shift + W        Отменить выделение
Ctrl + Shift + J        Присоединить линию
Ctrl + Shift + U        Переключить регистр
Ctrl + WhiteSpace       Автокомплит
Ctrl + Shift + WSpace   Активный автокомплит
Ctrl + Shift + Enter    Вставляет все необходимые завершающие символы
