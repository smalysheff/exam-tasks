Навигация проекту
Alt+F7          Найти все места где используется метод, класс, интерфейс
Ctrl+E          Недавние файлы
Ctrl+Shift+E    Недавние местоположения

Навигация по классу
F2              Найти проблемные методы в классе
Ctrl+F12        Навигация по методам
Ctrl+G          Перейти на строку №
Ctrl+/-         Свернуть и развернуть блоки кода
Alt + ↓/↑       Следующий метод


Работа с текстом / рефакторинг
Ctrl+W          Выделить слово
Ctrl+Shift+W    Отменить выделение
Alt+J           Выделить несколько вхождений
Ctrl+Shift+J    Присоединить линию
Ctrl+Shift+U    Переключить регистр


Копирование
Shift+Ctrl+V    Буфер копирования
Ctrl+Shift+C    Копирование пути
Ctrl+Alt+Shift+C    Копирование ссылки на метод
